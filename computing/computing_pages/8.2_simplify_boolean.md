# Simplifying Boolean Expressions

## De Morgan's Laws

- ¬(A ﹀ B) == ¬A ︿ ¬B
- ¬(A ︿ B) == ¬A ﹀ ¬B

### Simplifying Expressions

1. X ︿ 0 = 0
2. X ︿ 1 = X
3. X ︿ X = X
4. X ︿ ¬X = 0
5. X ﹀ 0 = X
6. X ﹀ 1 = 1
7. X ﹀ X = X
8. X ﹀ ¬X = 1
9. ¬¬X = X

## More rules

commutative rules

10. X ︿ Y = Y ︿ X
11. X ﹀ Y = Y ﹀ X

Associative rule

12. X ︿ (Y ︿ Z) = (Z ︿ Y) ︿ Z
13. X ﹀ (Y ﹀ Z) = (Z ﹀ Y) ﹀ Z

Distributive rule

14. X ︿ (Y ﹀ Z) = (X ︿ Y) ﹀ (X ︿ Z)
15. (X ﹀ Y) ︿ (W ﹀ Z) = (X ︿ W) ﹀ (X ︿ Z) ﹀ (Y ︿ W) ﹀ (Y ︿ Z)

absorption rules

16. X ﹀ (X ︿ Y) = X
17. X ︿ (X ﹀ Y) = X
